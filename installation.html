Installation

The recommended way for installing the Eclipse Jetty Plugin is via the Eclipse Marketplace. But you can install the plugin with the Eclipse install manager, too:

  1. Click Help | Install New Software ...
  2. Click on Add... next to the site
  3. Add repository: Eclipse Jetty - http://eclipse-jetty.sourceforge.net/update/
  4. Select the feature and click through rest of wizard
